Multi-Tab SSH Terminal

Full VTE terminal emulator with 256-color and truecolor support. Multiple concurrent sessions with instant tab switching.

🔒

Encrypted Credential Vault

AES-256-GCM encryption with Argon2id key derivation. Your passwords and SSH keys are encrypted at rest — never stored in plaintext.

📈

Real-Time Server Monitoring

CPU load, memory, disk usage, per-interface network traffic, and top processes — updated every 3 seconds via a dedicated SSH channel.

📁

SFTP File Browser

Browse remote directories, upload and download files with real-time progress bars. Quick-edit text files (JSON, YAML, configs) directly in the app.

Security First

Your credentials never leave your machine unencrypted

🛡

Two-Layer Encryption

Master Password → Argon2id (64MB, 3 passes) → KEK → AES-256-GCM → DEK → Your Data

  • Master password never stored — only the derived key is used
  • Each connection encrypted with unique nonce (IV)
  • Vault file unreadable without correct master password
  • Dedicated SSH session for monitoring — no interference with terminal
  • Private keys encrypted at rest, decrypted only in memory during use
  • Open source — audit the code yourself
