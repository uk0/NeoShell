Trust model

Your credentials, your machine.

Two-layer envelope encryption. Nothing leaves your device unencrypted.

🛡

Password → Argon2id (64MB, 3 passes) → KEK
KEK → AES-256-GCM → DEK → Your Data

  • Master password never stored — only the derived key is used
  • Each connection encrypted with a unique random nonce
  • Vault file is binary noise without the correct password
  • Dedicated exec SSH session — no lock contention with terminal
  • Private keys encrypted at rest, decrypted only in memory
  • Closed-source proprietary — no code exposure risk

History

Changelog

What shipped and when.

0.6.12 2026-04-23 Latest
Fixed
  • Win11 font rendering — primary UI font switched from Microsoft YaHei (inconsistent family names and malformed mstmc.ttf on some Win11 installs) to Segoe UI; CJK falls back automatically via cosmic-text glyph-level fallback
  • top/htop/nmon column drift — zero-width chars (combining marks, ZWJ, variation selectors, bidi, BOM) no longer advance cursor; they accumulated 1-cell drift per row
  • CJK glyphs rendered at 1.1x monospace size (was 1.3x) so wide chars stay within 2-cell slot and don't squeeze neighbors
  • is_wide_char coverage extended: Hangul Jamo, CJK Compat Forms, main Emoji ranges
0.6.11 2026-04-21
Added
  • Monitoring, SFTP, and file browsing now work on minimal-mode SSH (MINA SSHD, embedded gateways, bastion-behind-jumphost) — exec requests share the main session with mutex serialization
0.6.10 2026-04-21
Fixed
  • Minimal-mode SSH path for Apache MINA SSHD, embedded devices, and proprietary stacks — skip setenv/keepalive/exec-based shell start, use standard shell request
  • No more Read error: transport read infinite loop on non-OpenSSH servers
  • Log file auto-recreates if deleted mid-run
0.6.9 2026-04-21
Fixed
  • Bastion / embedded-SSH Failed to open channel: Session(-13) — tmux probing moved off main session; non-OpenSSH banner goes straight to RawShell
0.6.7 2026-04-21
Fixed

  • Two-phase handshake retry (filtered algo list → libssh2 defaults) covers kex-strict edge cases
  • Pre-handshake banner peek — failed connections now log the server's SSH version string
Added
  • CI algorithm-support verification — every platform build runs cargo test algorithm_tests; missing curve25519 / ecdh / ed25519 fails the build before shipping
  • Startup self-check logs libssh2 algorithm self-check: OK
